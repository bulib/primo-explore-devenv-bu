SEND FEEDBACK VIA THIS FORM Feature: User Tags

Users can add tags to individual records. This is available in the full record display as "Tags". If you haven't Signed In, the system will prompt you to before being able to add a tag.

Once added, the tag is visible on that record, and clicking will return all records with that tag.

It's also possible to pre-limit the search to within user tags via the basic or advanced search drop downs.

Note: tags are not part of the keyword index. They can only be searched using basic and advanced search drop downs.

This is different from My Favorites in that tags are visible to the public and indexed in user tags field.

When signed in the user who created the tags can delete them.

Some potential uses:

  • Create a link to a group of readings
  • Quickly sharing lists
  • When a pre-filtered "deep" link is not possible based on available metadata.

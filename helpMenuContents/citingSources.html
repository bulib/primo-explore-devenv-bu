Citation Formatting

Use the quotation mark icon to copy a citation in these formats:
  • MLA (7th edition)
  • MLA (8th edition)
  • Chicago/Turabian (16th ed.)
  • APA (6th ed.)
  • Harvard

Reference Managers

These are helpful for 1) exporting citations and managing resources as you're gathering them, and 2) generating footnotes or a bibliography for a paper.

BU Libraries Search can export individual or sets of records in formats for:

  • RefWorks
  • Zotero (.ris format)
  • Mendeley (.ris format)
  • Endnote
  • Easybib
  • Bibtex

See our Create Bibliographies guide for more details on selecting and using these reference managers.
